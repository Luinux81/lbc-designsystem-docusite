Intro

¡Bienvenido al manual de LBC Design System! Nuestro sistema de diseño es una colección integral de componentes, tipografía y colores que funcionan juntos a la perfección para crear una experiencia consistente y visualmente agradable para nuestros usuarios.

Componentes​

Una de las características únicas de nuestro sistema de diseño es su clasificación de componentes en cuatro modelos distintos: átomos, moléculas, organismos y páginas. Este modelo ayuda a organizar nuestros componentes de una manera que los hace fáciles de entender y usar.

  • Los átomos son los componentes básicos de nuestro sistema de diseño y consisten en los componentes más pequeños y simples, como botones, etiquetas e íconos.
  • Estos átomos se pueden combinar para formar moléculas, que son componentes más complejos que tienen un propósito específico, como formularios, encabezados o menús.
  • Los organismos son componentes aún más complejos que se componen de varias moléculas y átomos y se pueden reutilizar en todas nuestras aplicaciones, como una barra de navegación o una barra de búsqueda.
  • Finalmente, las páginas son representaciones completas de nuestras aplicaciones que reúnen todos los átomos, moléculas y organismos en una interfaz de usuario cohesiva y funcional. Nuestro sistema de diseño proporciona una variedad de páginas prediseñadas que se pueden usar como plantillas para nuevos proyectos, lo que garantiza un proceso de diseño uniforme y eficiente.

Tipografía y colores​

Además de la clasificación de los componentes, nuestro sistema de diseño también incluye una tipografía y una paleta de colores cuidadosamente seleccionadas, diseñadas para funcionar en armonía con los componentes. La tipografía se elige para que sea legible y estéticamente agradable, y la paleta de colores incluye una gama de tonos y sombras que se pueden usar para crear interfaces atractivas y visualmente atractivas.

Esperamos que este manual lo ayude a comprender y utilizar nuestro sistema de diseño en todo su potencial, creando interfaces de usuario atractivas y funcionales para todas sus aplicaciones.

Ejemplos de componentes y organización​

Sure, here are some examples of components for each category in a design system:

  1. Atoms:
    • Button
    • Slider
    • Icon
    • Typography
      • Typography scale
      • Fonts
    • Color palette
    • Grid system
    • Spacer
    • Switch
    • Fields
      • Text input field
      • Password input field
      • Textarea
      • Select dropdown
      • Checkbox
      • Radio button
    • Progress bar
    • Loading spinner
    • Tooltip
    • Popover
    • Modal backdrop
    • Badge
    • Label
    • Heading
    • Subheading
    • Paragraph
    • Quote
    • Code snippet
    • Link
    • Image
    • Video
    • Audio
  1. Molecules:
    • Search bar (combining text input field and button)
    • Navigation bar (combining buttons and typography)
    • Form field (combining label, input field, and validation message)
    • Modal (combining overlay, header, body, and footer)
    • Card (combining image, title, description, and button)
    • Breadcrumbs (combining typography and icons)
    • Dropdown menu (combining button and list)
  2. Organisms:
    • Header (combining logo, navigation bar, and search bar)
    • Footer (combining social icons, copyright information, and legal links)
    • Product list (combining card components)
    • Feature list (combining image, title, and description components)
    • User profile (combining avatar, name, email, and bio components)
    • Shopping cart (combining product list, subtotal, and checkout button components)
  3. Templates:
    • Landing page (combining header, hero section, feature section, testimonials section, and footer components)
    • Product detail page (combining header, image gallery, product description, reviews section, and related products section components)
    • Checkout page (combining header, order summary, shipping address form, payment form, and confirmation page components)
    • Blog post (combining header, author information, post content, related posts, and comments section components)
  4. Pages:
    • Home page (using the landing page template and customizing the content)
    • About page (using the organism components to create a team member section and company history section)
    • Contact page (using the molecule components to create a contact form and location map)
    • FAQ page (using the organism components to create a collapsible question and answer section)
